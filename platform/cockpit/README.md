# BlackGold Platform Cockpit

Static branded executive shell for the Lyons Command Center.

## Run
```bash
python -m http.server 8537 --directory static
# open http://localhost:8537/index.html
```

## Screenshots
```bash
python ../../../../AppData/Local/hermes/cache/scripts/screenshot_cockpit.py
```

## Data
Place JSON payloads in `static/data/` to feed live dashboard surfaces.

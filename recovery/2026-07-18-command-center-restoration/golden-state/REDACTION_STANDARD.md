# REDACTION STANDARD

Never log:
- Full Telegram bot token
- API keys
- Bearer tokens
- Passwords
- Private headers
- Full confidential payloads
- Unredacted banking data
- Private user content unless operationally required and authorized

Allow:
- Masked identifiers and last-4 references
- Error categories and status codes
- File paths and timestamps
- Request metadata excluding secrets
- Incident IDs and correlation IDs
